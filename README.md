# Srvpru

Srvpru is an alternative for srvpro, rewritten by Rust.

## It supports:
* room
* plugin
* broadcast
* hot update

## Don't support:
* chatroom

## Example plugin
* debugger
* version check
* mycard log on
* lock deck
* relink
* chat sensitive words check
* upload deck to analyzer
* upload duel result to analyzer
* cloud replay
